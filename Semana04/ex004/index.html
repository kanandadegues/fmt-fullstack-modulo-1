Exercício 04

  1. Usando seu entendimento, explique o que cada um dos métodos de array abaixo executam:
    • forEach
    • map
    • filter
    • find
    • reduce
    • every

Resposta

forEach: executa uma dada função em cada elemento de um array.

map: executa a função para cada item e cria um novo array com todos returns.

filter: cria um novo array com os itens que passarem no teste da função fornecida.

find: retorna o valor do primeiro item que satisfizer a função de teste provida.

reduce: executa uma função reduce(fornecida por você) para cada elemento do arry, resultando em um único valor de retorno.

every: testa se todos os itens passam pelo teste implementado pela função fornecida.
