Exercício 03

  1. Conforme vimos nas aulas a Programação Orientada a Objetos é um paradigma de programação. Com suas palavras descreva o conceito do paradigma em questão e por que usamos classes em Javascript para aplicar este no desenvolvimento de uma aplicação

Resposta

POO é baseado na construção de estruturas que agrupam caracterpisticas e/ou comportamentos, que são chamado de objetos. Sabendo que classe é um tipo de objeto, o POO utiliza classes para agrupar coisas com caracteristicas similares ou que façam parte de um mesmo contexto, para que o código fique mais limpo, organizado e de fácil entendimento.
