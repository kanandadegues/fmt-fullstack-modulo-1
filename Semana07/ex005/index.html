Exercício 05

  1. Uma aplicação Angular possui uma versão desatualizada da dependência "@ng-bootstrap/ng-bootstrap". Utilizando o NPM CLI, qual o comando responsável por instalar a versão 13.1.1 dessa dependência?

Resposta

O comando responsável ppela instalação da versão 13.1.1 é: npm i @ng-bootstrap/ng-bootstrap@13.1.1
