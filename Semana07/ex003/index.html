Exercício 03

  1. Qual/quais comandos podem ser utilizados para criar um projeto localmente?
    Qual/quais comandos podem ser utilizados para executar um projeto localmente?

Resposta

O comando que pode ser utilizado para criar um projeto localmente é o ng new (cria uma nova pasta contendo uma estrutura de diretórios com os arquivos necessários par ainiciar um projeto no angular).
Os comandos utilizados para executar o projeto localmente é o ng serve (gera um serivdor http na máquina para poder testar, de forma fácil e imediata, as alterações realizadas nos arquivos dos projetos) e o npm start.
