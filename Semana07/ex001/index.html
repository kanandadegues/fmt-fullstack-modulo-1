Exercício 01

  1. Defina o que é Angular e quais suas características.

Resposta

O Angular é um framework utilizado para a criação de Single-Page Applications (SPA), que é um aplicação web consumida em uma única página. Ele suporta Javascript e Typescript, a sua arquitetura é baseada em componentes, isso torna o desenvolvimento da aplicação mais flexivel pois facilita a reutilização de códigos sem a necessidade de reescrevê-los, possui CLI (linhas de comando para instrução direta). É mais rápido que o Angular JS, além de suportar injeção de dependência e navegadores mobile.
