Exercício 04

  1. Dada uma aplicação utilizando a versão 14 do Angular, consulte o NPM e responda:
    • Qual a versão mais recomendada para instalação do ng-bootstrap?
    • Qual a versão mais recomendada para instalação do Bootstrap (CSS)?
    • Há alguma outra dependencia recomendada para instalação?

Resposta

A versão mais recomendada para a instalação do ng-bootstrap é a 13.x.x e a versão mais recomendada para a instalação do Bootstrap é a 5.2.0. A dependencia recomendada é ter o popper na versão 2.10.2.
