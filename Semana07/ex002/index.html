Exercício 02

  1. O que é Angular CLI e quais suas vantagens?

Resposta

O Angular CLI é uma ferramenta de interface de linha de comandos que é utilizado para iniciar, desenvolver e manter aplicativos Angular. Ela é a forma mais fácil, rápida e recomendada de se iniciar uma aplicação. Basicamente o Angular CLI cria toda a estrutura necessária para o projeto, com arquivos, diretórios e scripts necessários para o desenvolvimento da aplicação. Além da configuração do ambiente, a ferramenta também permite executar diversas atividades no ciclo de desenvolvimento como: fazer a compilação da aplicação, adicionar módulos para os componentes, preparar a aplicação para o servidor e gerar o pacote final com os arquivos minificados e otimizados.
