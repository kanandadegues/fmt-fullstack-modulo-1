Exercício 02

  1. Explique brevemente o método “console.log()”. Explique a sintaxe do método console.log() e o que ele faz.

Resposta

O console.log é um método JavaScript utilizado para imprimir no console do navegador alguma informação. Em relação a sintaxe é necessário entender que o console é um objeto, e que log() é um método desse objeto, que tem como objetivo emitir mensagens de depuração do sistema. Normalmente utiliza-se o console.log para acompanhar os valores de variáveis durante a execução de algum código com o intuito de identificar algum bug ou entender como o código está se comportando.
