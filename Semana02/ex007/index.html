Exercício 07

  1. Crie um programa com JavaScript que a partir de um salário inicial, calcule quantos meses será necessário para atingir uma meta salarial recebendo um aumento a cada 6 meses equivalente a 10% no salário corrente. Para executar esse exercício você provavelmente vai precisar saber usar: estrutura de repetição, estrutura condicional, operador Remainder (%), operadores aritméticos (divisão, multiplicação e soma).

Resposta

A resolução do exercício conta com a utilização do window.prompt() para coletar as informações de salário e da meta salarial. Definiu-se também uma variável para o mês e para o novo salário. Utilizou-se a condicional DO, WHILE e IF para que pudessemos calcular a porcentagem de aumento a cada 6 meses (utilizando o remainder e os operadores aritméticos). Para finalizar utilizamos um window.alert() e um template string para informar ao usuário em quantos meses ele chegará na meta salarial informada.
