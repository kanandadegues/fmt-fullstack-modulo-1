Exercício 01

  1. Com suas palavras descreva a relação entre ECMAScript e JavaScript. Escreva da maneira que você considerar mais fácil a diferença entre as linguagens Javascript e ECMAScript.

Resposta

O ECMAScript é a padronização da linguagem JavaScript, feito pela ECMA (European Computer Manufacturers Association). A diferença entre as duas linguagens é que a ECMAScript é uma especificação de linguagem, ou seja, ela define os padrões para uma linguagem de programação, e o JavaScript é a implementação desses padrões.
